# Banking

The first step is to set up a Financial Institution. In the menu, select Banking / Financial Institutions, then click the New button. Here you can add a new bank, credit card, or line of credit.

{% hint style="info" %}
The type of institution is determined by the type of general ledger account you choose.
{% endhint %}
