# Accounting Periods

<figure><img src="../.gitbook/assets/PCP4B.jpg" alt=""><figcaption></figcaption></figure>

Once on the accounting period setup screen, click the button labelled **Generate**.

<figure><img src="../.gitbook/assets/image (12).png" alt=""><figcaption><p>P4 Books Generate Posting periods</p></figcaption></figure>

On this screen you can enter the name (e.g. 2023), start date, end date, and type. Once the details are complete, click the Submit button to create the accounting periods in P4 Books.

{% hint style="info" %}
Once an accounting period is locked, transactions cannot be posted to it unless the period is unlocked by your accounting team.
{% endhint %}
